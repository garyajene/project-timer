# Project Timer

A dark, minimal personal productivity web app UI foundation for a single user. Version 1 focuses on reusable vanilla JavaScript rendering helpers and polished responsive layouts; timer logic, persistence, sounds, notifications, authentication, and multi-user functionality are intentionally out of scope.

## Scripts

- `npm run dev` - start a local static development server.
- `npm run build` - create a production build.
- `npm run preview` - preview the production build locally.
